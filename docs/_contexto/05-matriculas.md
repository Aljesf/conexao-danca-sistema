# 📘 Contexto — Matrículas

## 1. O que é o domínio

Matrícula é:

- vínculo formal
- base pedagógica
- origem financeira
- origem documental

## 2. Estado atual real

Matrícula conecta:

- pessoa
- turma
- financeiro

Ela:

- gera cobranças
- pode gerar entrada (pró-rata)
- integra com conta interna

Características:

- matrícula não define vencimento
- matrícula não é cobrança
- matrícula gera eventos financeiros

## 3. Bolsa

Nome oficial de negócio:

👉 `Bolsa`

Situação atual:

- sistema ainda pode conter nomenclatura antiga
- conceito já consolidado institucionalmente

Regras:

- não altera valor contratado
- altera forma de liquidação
- deve ser rastreável

## 4. Problemas atuais

- confusão entre matrícula e financeiro
- nomenclatura antiga ainda presente
- inconsistência em telas

## 5. Direção institucional

Padronizar:

- matrícula como ato contratual
- financeiro separado
- `Bolsa` como vínculo institucional

Separações obrigatórias:

- matrícula ≠ cobrança
- matrícula ≠ pagamento
- bolsa ≠ desconto automático

## 6. Regras para o Codex

- nunca tratar matrícula como título financeiro
- nunca definir vencimento na matrícula
- sempre usar `Bolsa` em novos contextos
- manter vínculo com `Conta Interna`
